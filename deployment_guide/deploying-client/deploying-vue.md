# Client-Server 연동

## Server

- CORS whitelist 추가

    ```python
    CORS_ALLOWED_ORIGINS = [
        "S3 버킷 엔드포인트 주소",
    ]
    ```

## Client

- API 요청 주소 수정

    (아래 코드는 예시입니다. 각 프로젝트의 상황에 맞게 바꿔주시면 됩니다.)

    ```js
    const BASE_URL = 'http://배포된_서버_도메인_또는_PUBLIC_IP'
    axios.get(`${BASE_URL}/api/movies/`)
    .then(..)
    .catch(...)
    ```
